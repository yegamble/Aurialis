## Summary

**Goal:** Create an immersive mode compatible with AirPods head tracking, similar to Apple's Spatial Audio with Dynamic Head Tracking (https://support.apple.com/en-gb/guide/iphone/iphac459a29e/ios). The mode should detect head orientation from AirPods and rotate the audio scene accordingly so that audio sources appear fixed in space relative to the listener.
